# Army Board AI Study Assistant

This is a beginner AI-assisted Python project designed to help Soldiers study for Army promotion boards.

## What it does
The program asks the user for an Army study topic and creates:
- a simple explanation
- study tips
- quiz questions

## Why I built it
I built this project to practice Python, automation, and AI-assisted software development while creating something useful for military education and board preparation.

## Technologies Used
- Python
- AI-assisted development
- GitHub
